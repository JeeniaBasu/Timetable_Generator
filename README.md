# Timetable Generator

## Overview
A Python CLI tool developed for the Smart India Hackathon 2025 (SIH 25) that generates a weekly timetable for students.  
Users can input subjects, timings, and priorities, and get an organized schedule.

## Features
- Create weekly timetables with multiple subjects
- Set priorities for important subjects
- Display timetable in a clear, readable format
- Optionally save timetable for future reference

## Tech Stack
- Python 3
- Standard Python libraries (datetime, etc.)
- Command Line Interface (CLI)

## Workflow
- The program prompts the user to enter subjects and time slots.
- Checks for conflicts and arranges subjects based on priority.
- Outputs a weekly timetable in a readable format.
